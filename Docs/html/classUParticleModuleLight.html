bool UParticleModuleLight::bAffectsTranslucency

Whether lights from this module should affect translucency. Use with caution. Modules enabling this should only make a few particle lights at most, and the smaller they are, the less they will cost.

◆ InverseExposureBlend

float UParticleModuleLight::InverseExposureBlend = 0.0f
private

Blend Factor used to blend between Intensity and Intensity/Exposure. This is useful for gameplay lights that should have constant brighness on screen independent of current exposure. This feature can cause issues with exposure particularly when used on the primary light on a scene, as such it's usage should be limited.
