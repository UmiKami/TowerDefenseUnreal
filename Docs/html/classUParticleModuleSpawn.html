Retrieve the spawn amount this module is contributing. Note that if multiple Spawn-specific modules are present, if any one of them ignores the SpawnRate processing it will be ignored.

Parameters
OwnerThe particle emitter instance that is spawning.
OffsetThe offset into the particle payload for the module.
OldLeftoverThe bit of timeslice left over from the previous frame.
DeltaTimeThe time that has expired since the last frame.
NumberThe number of particles to spawn. (OUTPUT)
RateThe spawn rate of the module. (OUTPUT)
Returns
bool false if the SpawnRate should be ignored. true if the SpawnRate should still be processed.
