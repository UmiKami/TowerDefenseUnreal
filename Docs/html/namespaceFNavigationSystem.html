◆ AddNavigationSystemToWorld()

ENGINE_API void FNavigationSystem::AddNavigationSystemToWorld ( UWorld & WorldOwner,
const FNavigationSystemRunMode RunMode = FNavigationSystemRunMode::InvalidMode,
UNavigationSystemConfig * NavigationSystemConfig = nullptr,
const bool bInitializeForWorld = true,
const bool bOverridePreviousNavSys = false )

Creates an instance of NavigationSystem (class being specified by WorldSetting's NavigationSystemConfig) A new instance will be created only if given WorldOwner doesn't have one yet. The new instance will be assigned to the given WorldOwner (via SetNavigationSystem call) and depending on value of bInitializeForWorld the InitializeForWorld function will be called on the new NavigationSystem instance. (

See also
UWorld.NavigationSystem)
Parameters
RunModeif set to a valid value (other than FNavigationSystemRunMode::InvalidMode) will also configure the created NavigationSystem instance for that mode
NavigationSystemConfigis used to pick the navigation system's class and set it up. If null then WorldOwner.WorldSettings.NavigationSystemConfig will be used
