Note that this is called at load on the UAnimInstance CDO to avoid needing to call this at runtime. This is called on the game thread.

◆ bForwardRequestsThroughSkippedCachedPoseNodes

bool FAnimNode_Inertialization::bForwardRequestsThroughSkippedCachedPoseNodes = true
private

When enabled this option will forward inertialization requests through any downstream UseCachedPose nodes which have had their update skipped (e.g. because they have already been updated in another location). This can be useful in the case where the same cached pose is used in multiple places, and having an inertialization request that goes with it caught in only one of those places would create popping.
