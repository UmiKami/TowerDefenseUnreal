◆ bContributeToMass

uint8 FKShapeElem::bContributeToMass
private

True if this shape should contribute to the overall mass of the body it belongs to. This lets you create extra collision volumes which do not affect the mass properties of an object.

◆ CollisionEnabled

TEnumAsByte<ECollisionEnabled::Type> FKShapeElem::CollisionEnabled

Course per-primitive collision filtering. This allows for individual primitives to be toggled in and out of sim and query collision without changing filtering details.
