◆ CommitMeshDescription()

ENGINE_API void FSkeletalMeshSourceModel::CommitMeshDescription ( bool bInUseHashAsGuid)

Commits the currently cached mesh description (as retrieved via GetMeshDescription or TryGetMeshDescription) into bulk storage so that it will be persisted.

Parameters
bInUseHashAsGuidIf set to true, then the DDC hash will be computed from the contents of the mesh itself, ensuring that if the same mesh is committed again, the DDC will not be updated. If set to false, then a new, random GUID is generated each time, causing the DDC to be updated regardless of whether the mesh changed or not.

void FSkeletalMeshSourceModel::ConvertRawMeshToMeshDescriptionBulkData ( )
private

Convert already stored old raw import format to mesh description and commit to bulk data. The new bulk data will default to using the mesh hash as the GUID, so that any future commits on an unchanged mesh will not cause a DDC invalidation.
