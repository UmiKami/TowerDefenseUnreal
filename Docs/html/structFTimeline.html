◆ PlayingStateTracker

uint8 FTimeline::PlayingStateTracker
private

If the first bit is set to 1 (PlayingStateTracker & 0x01 == 1), then we are playing

The rest of the bits in the uint8 are reserved for keeping track of the "dirty" state, being incremented when our state is modified. This ensures that the value is replicated if it changes multiple times in one frame, such as calling "Play From Start" in the resulting "Finished" delegate.

You should modify this value using the "ChangeMarkPlayingState" function.
