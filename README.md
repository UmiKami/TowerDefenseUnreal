# UNREAL TOWER DEFENSE

I could not think of a better game. Basic concept, choose and place tower, tower goes pew pew.

# MVP

- Player selects a tower on toolbar UI.
- Player Places tower on the world.
- Tower shoots at nearest enemies within certain range.
- Enemies are Spawned on point A and move toward point B
- Enemy that collides with point B subtract player health.
- Health reaches 0, player losses.
- **GOAL**: Survive as long as you can.
